⏰ Automatischer Abruf
Wie funktioniert der tägliche Abruf?
WanderSuite prüft automatisch jeden Morgen um 07:00 Uhr alle aktiven Tracker. Die Zeitzone ist in den Einstellungen konfigurierbar (Standard: Europe/Rome). Du kannst jeden Tracker auch jederzeit manuell mit dem ⟳ Jetzt-Button aktualisieren.
✈️ IATA-Codes
Was sind IATA-Codes?
IATA-Codes sind 3-buchstabige Kürzel für Flughäfen — international standardisiert. Beispiele: BZO = Bozen, VIE = Wien, BCN = Barcelona, LHR = London Heathrow.
🔍 IATA-Code nachschlagen: iata.org/code-search oder einfach „{Stadt} Flughafen IATA" googeln.
🔑 API-Keys
Wo bekomme ich einen SerpAPI-Key?
SerpAPI wird für Google Flights und Booking-Tracker benötigt. Das kostenlose Paket beinhaltet 100 Suchen/Monat — für den normalen Gebrauch ausreichend.
🌐 serpapi.com → Konto erstellen → Dashboard → API Key kopieren
Wo bekomme ich einen Google Gemini-Key?
Gemini wird für KI-Reiseempfehlungen unter Inspiration verwendet. Das kostenlose Kontingent (Gemini 1.5 Flash) ist für normale Nutzung völlig ausreichend.
🌐 aistudio.google.com → „Get API key" → Neuen Key erstellen
Wo bekomme ich einen OpenAI-Key?
OpenAI ist eine Alternative zu Gemini. WanderSuite nutzt gpt-4o-mini — günstig und schnell.
🌐 platform.openai.com → API Keys → Create new secret key
Wo werden die Keys gespeichert?
Lokal im Browser (localStorage) und optional verschlüsselt auf deinem Backend (AES-Fernet). Beim Speichern in den Einstellungen werden sie ans Backend übertragen und dort codiert in SQLite gespeichert — niemals im Klartext.
🔔 Benachrichtigungen
Wie richte ich Telegram-Benachrichtigungen ein?
WanderSuite sendet dir eine Nachricht, sobald ein Flugpreis sinkt. Dazu brauchst du zwei Dinge:

1. Telegram Bot erstellen: Öffne Telegram → suche @BotFather → tippe /newbot → folge den Anweisungen → kopiere den Bot Token.

2. Chat-ID ermitteln: Schreibe deinem Bot eine Nachricht, dann öffne @userinfobot oder @myidbot — sie zeigen deine numerische Chat-ID.
⚙️ Einrichten: Einstellungen → 🔔 Alerts → Telegram → Token + Chat-ID eingeben → "Testen"
Wie richte ich Gotify-Benachrichtigungen ein?
Gotify ist ein self-hosted Push-Benachrichtigungsdienst — perfekt für Unraid. Dazu brauchst du:

1. Gotify-Server: Läuft bereits auf deinem Unraid? Dann hast du eine URL wie http://192.168.1.51:8070.

2. App Token: In Gotify → Apps → Create Application → Token kopieren.
🌐 gotify.net — Docker Image: gotify/server
⚙️ Einrichten: Einstellungen → 🔔 Alerts → Gotify → URL + Token → "Testen"
🎯 Wie der Preis-Radar funktioniert
Direktes Scraping vs. SerpAPI — was ist der Unterschied?
Direktes Scraping Ryanair & Homair: WanderSuite spricht direkt mit der Ryanair-API bzw. Homair. Kein externer Key nötig — dafür können Anti-Bot-Maßnahmen gelegentlich Abrufe blockieren.

SerpAPI Google Flights & Booking: Die Preise werden über die SerpAPI-Schnittstelle abgerufen, die Google-Suchergebnisse strukturiert zurückgibt. Zuverlässiger, aber auf 100 Anfragen/Monat (Free Tier) limitiert.
⏸ Tracker-Status
Was bedeutet "Pausiert"?
Ein pausierter Tracker wird beim automatischen Morgen-Abruf übersprungen, aber nicht gelöscht. Du kannst ihn jederzeit mit dem ▶-Button wieder aktivieren. Nützlich für abgeschlossene Reisen, die du noch im Verlauf behalten möchtest.
Was bedeutet "Geblockt"?
Das Backend hat einen temporären Fehler beim Abrufen erhalten — meist durch Anti-Bot-Maßnahmen bei Ryanair. Das ist normal und temporär. Der nächste automatische Abruf (07:00 Uhr) versucht es erneut. Du kannst auch sofort mit ⟳ erneut versuchen.
Was bedeutet das ~-Zeichen ("Schätzwert")?
Wenn die Gepäckpreise nicht direkt vom Scraper zurückgegeben werden, schätzt WanderSuite sie anhand historischer Daten. Der Gesamtpreis ist dann eine Annäherung, kein garantierter Wert. Das Badge ~Schätzwert zeigt dies an.
Wie viele Tracker kann ich anlegen?
Unbegrenzt — WanderSuite ist self-hosted, also kein künstliches Limit. Beachte nur das SerpAPI-Kontingent (100/Monat) für Google Flights und Booking-Tracker.
📓 Reisetagebuch — Dawarich-Integration
Was ist Dawarich und wie funktioniert der Sync?
Dawarich ist eine self-hosted App, die deinen GPS-Standortverlauf aufzeichnet (z. B. von OwnTracks oder Overland). WanderSuite analysiert diese Daten und erkennt automatisch Übernachtungsreisen.
Wie erkennt WanderSuite Reisen automatisch?
Der Algorithmus funktioniert in 5 Schritten:

1. Alle GPS-Punkte werden von Dawarich abgerufen (paginiert)
2. Haversine-Distanz zum Heimatort wird berechnet
3. Nur Punkte > 50 km von zu Hause werden behalten
4. Tage mit mind. 2 aufeinanderfolgenden Nächten weg = eine Reise
5. Reverse Geocoding via OpenStreetMap Nominatim (kein API-Key nötig)
⚙️ Heimatkoordinaten einstellen: Einstellungen → Integrationen → Dawarich → Latitude / Longitude
💶 Reisebudget — ActualBudget-Integration
Was ist ActualBudget und wie funktioniert der Sync?
ActualBudget ist eine self-hosted Budgetierungs-App. WanderSuite importiert deine Reise-Kategorien daraus und zeigt Ausgaben direkt im Budget-Tab an.
⚙️ Einrichten: Einstellungen → Integrationen → ActualBudget → Server-URL, Passwort und Budget-Name eingeben. Dann Reise-Kategorien als kommagetrennte Liste hinterlegen (z.B. "Urlaub, Flüge").
Kann ich das Budget auch ohne ActualBudget nutzen?
Ja. Über den manuellen Modus kannst du Reisen mit Namen, Datum und Kosten direkt in WanderSuite eintragen. Diese werden lokal im Browser gespeichert (kein Backend nötig).
🗺️ Wunschziele (Bucket List)
Wie funktionieren die Wunschziele?
Die Wunschziele sind vollständig lokal im Browser gespeichert — kein Backend, kein Account nötig. Trage einfach ein Ziel (z.B. "Kyoto, Japan") und ein Wunschdatum (z.B. "Sommer 2027") ein. WanderSuite wählt automatisch ein passendes Reise-Emoji.
✨ KI-Reiseempfehlungen
Wie nutze ich die KI für Reise-Inspiration?
Gib unter Inspiration einfach eine Beschreibung dessen ein, was du suchst — z.B. "Warmes Meer im Oktober, Kultur, kein Massentourismus". WanderSuite schickt deine Anfrage an Gemini oder OpenAI und zeigt konkrete Ziele mit Reisezeit, Budget-Einschätzung und Top-Highlights.
Welchen KI-Anbieter soll ich wählen?
Empfohlen Google Gemini — kostenlos, schnell, gute Reise-Kenntnisse. API-Key auf aistudio.google.com.

Alternative OpenAI gpt-4o-mini — kostenpflichtig, aber sehr günstig (Cent-Beträge pro Anfrage). Gut für komplexere Anfragen.
⚙️ KI-Anbieter wählen: Einstellungen → APIs & KI → Anbieter auswählen + Key eintragen
Was macht der "Dawarich-Verlauf nutzen"-Toggle?
Wenn du Dawarich verbunden hast, kann WanderSuite Orte, an denen du schon warst, aus den Empfehlungen ausschließen. So bekommst du wirklich neue Vorschläge statt Länder zu sehen, die du schon kennst. Der Toggle aktiviert diese Filterung.
Werden meine Anfragen gespeichert?
Nein. Die Anfragen werden direkt von deinem Backend an den KI-Anbieter geschickt und nicht gespeichert. WanderSuite ist vollständig self-hosted — deine Daten gehören dir.
